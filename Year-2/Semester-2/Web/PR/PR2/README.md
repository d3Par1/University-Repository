# Практична робота №2 — JavaScript. Динамічний контент

## Моніторинг насосної станції водопостачання

**Курс:** Основи Веб-програмування
**Варіант:** 15 — Насосна станція водопостачання
**Група:** ТВ-43
**Семестр:** 2-й, 2025/26 н.р.

## Опис

Інтерактивна веб-сторінка моніторингу насосної станції водопостачання з динамічним оновленням параметрів за допомогою JavaScript. Реалізовано ручне та автоматичне оновлення даних, візуальну індикацію статусів, графіки Chart.js та журнал вимірювань.

## Структура проекту

```
PR2/
├── index.html          — головна сторінка (моніторинг)
├── styles.css          — CSS-стилі (водна тематика)
├── script.js           — JavaScript логіка
├── Звіт_ПР2.docx      — звіт до практичної роботи
└── README.md           — опис проекту
```

## Функціональність

### Основні параметри моніторингу
- **Тиск на виході** (3–6 атм, норма: 4–5.5 атм)
- **Продуктивність** (50–200 м³/год, норма: 80–150 м³/год)
- **Споживана потужність** (0–100 кВт, норма: 30–80 кВт)
- **Рівень води в резервуарі** (0–10 м, норма: 3–8 м)

### Додаткові параметри
- Кількість працюючих насосів (4 індикатори)
- Частота обертання (прогрес-бар)
- Температура двигуна (статус)

### JavaScript функціонал
- Генерація випадкових даних (`getRandom`)
- Визначення статусу параметрів (`checkStatus`: normal / warning / danger)
- Ручне оновлення (кнопка «Оновити»)
- Автоматичне оновлення (таймер з вибором інтервалу)
- Графіки зміни параметрів у часі (Chart.js)
- Журнал вимірювань із збереженням у localStorage
- Експорт даних у CSV
- Годинник реального часу

## Технології
