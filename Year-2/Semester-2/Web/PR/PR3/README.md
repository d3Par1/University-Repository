## Опис

Розробити форму збору даних з серверною обробкою на Node.js + Express. Форма для реєстрації споживачів електроенергії або відстеження генерації.

**Вимоги:**
- Поля: назва, тип об'єкта, адреса, параметри підключення
- Валідація на клієнті та сервері
- Збереження даних у JSON або БД
- Серверна обробка POST-запитів

**Стек:** HTML, CSS, Node.js, Express, JSON/DB
