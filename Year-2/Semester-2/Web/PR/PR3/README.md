# Практична робота №3 — HTML-форми та серверна обробка (Node.js + Express)

## Реєстр резервних джерел живлення (ДГУ)

**Курс:** Основи Веб-програмування
**Варіант:** 15 — Резервні джерела (ДГУ)
**Група:** ТВ-43
**Семестр:** 2-й, 2025/26 н.р.

## Опис

Веб-додаток для реєстрації та обліку дизельних генераторних установок (ДГУ) на об'єктах критичної інфраструктури. Реалізовано з використанням Node.js, Express та збереженням даних у JSON-файл. Клієнтська частина побудована на HTML5, Bootstrap 5 та JavaScript (Fetch API).

## Структура проекту

```
PR3/
├── server.js              — серверний код (Express)
├── package.json           — залежності проекту
├── public/                — статичні файли
│   ├── index.html         — головна сторінка з формою
│   ├── styles.css         — CSS-стилі (енергетична тематика)
│   └── script.js          — клієнтський JavaScript
├── data/
│   └── generators.json    — файл збереження даних
└── README.md              — опис проекту
```

## Функціональність

### Форма реєстрації
- **Назва об'єкта** — текстове поле (обов'язкове)
- **Тип ДГУ** — вибір (дизельна, бензинова, газова, газотурбінна, портативна)
- **Потужність (кВА)** — числове поле
- **Тип палива** — вибір (дизель, бензин, природний газ, пропан)
- **Адреса розташування** — багаторядкове поле
- **Витрата палива (л/год)** — автоматичний розрахунок за потужністю
- **Стан** — вибір (справний, потребує ТО, несправний, виведений)
- **Дати ТО** — останнє та наступне технічне обслуговування
- **Примітки** — необов'язкове текстове поле

### Серверна частина (API)
| Метод  | Маршрут                  | Опис                                  |
|--------|--------------------------|---------------------------------------|
| GET    | `/api/generators`        | Список ДГУ (з фільтрами та пошуком)  |
| GET    | `/api/generators/stats`  | Статистика реєстру                   |
| POST   | `/api/generators`        | Додати новий ДГУ                     |
| PUT    | `/api/generators/:id`    | Оновити існуючий ДГУ                 |
| DELETE | `/api/generators/:id`    | Видалити ДГУ                         |

### Додатковий функціонал
- Автоматичний розрахунок витрати палива за потужністю та типом палива
- Відстеження ТО: підсвічування протермінованих та найближчих обслуговувань
- Пошук за назвою або адресою
- Фільтрація за типом ДГУ та станом
- Редагування існуючих записів
- Валідація на клієнті (Bootstrap) та на сервері
- Захист від XSS (екранування виводу)
- Статистика: загальна потужність, середня витрата, кількість за статусами

## Технології

- **Сервер:** Node.js + Express 4
- **Зберігання:** JSON-файл (fs module)
- **Клієнт:** HTML5 (семантичні теги, ARIA), CSS3 (змінні, градієнти, анімації)
- **UI:** Bootstrap 5.3.0
- **JavaScript:** Fetch API, async/await, FormData, DOM manipulation

## Запуск

```bash
npm install
npm start        # або: node server.js
```

Сервер доступний за адресою: http://localhost:3000

Для режиму розробки (автоперезапуск):
```bash
npm run dev
```
