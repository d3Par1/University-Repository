# Практична робота №1 — HTML та CSS

## Інформаційний веб-сайт енергетичного об'єкта

**Курс:** Основи Веб-програмування
**Варіант:** 15 — Вітрова станція «Приазовська», 150 МВт
**Група:** ТВ-43
**Семестр:** 2-й, 2025/26 н.р.

## Опис

Односторінковий інформаційний сайт вітрової електростанції «Приазовська» (Запорізька обл.).
Реалізовано з використанням HTML5, CSS3 та Bootstrap 5.

## Структура проекту

```
PR1/
├── index.html          — головна сторінка
├── styles.css          — власні CSS-стилі
├── images/
│   ├── wpp-general.jpg — загальний вигляд ВЕС
│   ├── wpp-layout.jpg  — розташування турбін
│   └── wpp-scheme.svg  — однолінійна електрична схема
└── README.md           — опис проекту
```

## Розділи сайту

- **Про об'єкт** — історія, призначення, особливості ВЕС
- **Особливості та переваги** — екологічність, надійність, автоматизація
- **Технічні параметри** — таблиця з 10 характеристиками
- **Схема та вигляд** — фото розташування турбін + однолінійна схема
- **Контактна інформація** — адреса, телефон, email

## Технології

- HTML5 (семантичні теги: header, nav, main, section, footer)
- CSS3 (змінні, градієнти, анімації, медіа-запити)
- Bootstrap 5.3.0 (navbar, grid, cards, table, utilities)

## Запуск

Відкрити `index.html` у браузері.
