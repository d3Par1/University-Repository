# Практична робота №5 — Клієнтський веб-додаток моніторингу біогазової електростанції

**Курс:** Основи Веб-програмування
**Варіант:** 15 — Біогазова установка
**Група:** ТВ-43
**Семестр:** 2-й, 2025/26 н.р.

## Опис

Real-time дашборд для моніторингу мережі біогазових електростанцій України. Клієнтська частина (HTML5 + Bootstrap 5 + Chart.js + custom SVG) візуалізує параметри ферментації, склад біогазу та електричну потужність у реальному часі. Серверна частина (Node.js + Express + ws) симулює мезофільний анаеробний процес (оптимум 38–40°C) і транслює дані через WebSocket кожні 2 секунди.

## Архітектура (гібрид REST + WebSocket)

```
┌─────────────── БРАУЗЕР ───────────────┐
│ Chart.js · SVG gauges · HTML table    │
│            ▲                          │
│  ┌─── WebSocket /ws ─────┐ push 2с    │
│  └─── REST /api/history ─┘ initial    │
└────────────│──────────────────────────┘
             ▼
┌─────── Node.js (Express + ws) ────────┐
│   in-memory simulation tick() 2с      │
│   broadcast() to all WS clients       │
└───────────────────────────────────────┘
```

## Параметри об'єкта (Варіант 15)

| Параметр                        | Тип     | Опис                                   |
|---------------------------------|---------|----------------------------------------|
| `biogasFlow`                    | number  | Обсяг виробленого біогазу (м³/год)     |
| `methaneContent`                | number  | Вміст метану CH₄ (%)                   |
| `co2Content`                    | number  | Вміст CO₂ (%)                          |

| `fermenterTemp`                 | number  | Температура ферментера (°C)            |
| `substrateLoading`              | number  | Завантаження субстрату (т/добу)        |
| `efficiency`                    | number  | ККД ферментації (0..1)                 |
| `status`                        | string  | `running` / `warning` / `low_efficiency` |

## Endpoints API

| Метод | URL                                | Призначення                                  |
|-------|------------------------------------|----------------------------------------------|
| GET   | `/api/biogas-plants`               | Список усіх станцій з поточним станом        |
| GET   | `/api/biogas-plants/stats`         | Агрегована статистика по мережі              |
| GET   | `/api/biogas-plants/:id`           | Деталі однієї станції                        |
| GET   | `/api/history`                     | Останні 24 години (по 1 точці на годину)     |
| GET   | `/api/status`                      | Стан сервера, кількість WS-клієнтів          |
| WS    | `ws://host/ws`                     | Real-time push: snapshot + tick кожні 2с     |

## Технології

- **Сервер:** Node.js 20+, Express 4, ws (WebSocket)
- **Симуляція:** in-memory mesophilic fermentation model (CH₄ heat 6 кВт·год/м³, КГУ ηел 40%)
- **Клієнт:** HTML5, CSS3, Bootstrap 5.3, Chart.js 4.4, vanilla JS
- **Візуалізації:** Chart.js (line + doughnut), custom SVG gauges (3 теплові зони), HTML5 table

## Модель симуляції

- **Температура ферментера:** дрейф ±0.15°C/тік навколо 39°C, межі 37–42°C (мезофільний оптимум)
- **Метан CH₄:** 53–63% (типово 56–60%)
- **CO₂:** доповнення до 100% мінус H₂S та інші
- **H₂S:** 0.025–0.045% (250–450 ppm)
- **ККД ферментації:** `1 − |T − 39| × 0.08` (поза оптимумом — падає)
- **Електр. потужність:** `Q_газ × (CH₄/100) × 6 × 0.4` кВт
- **Накопичення енергії:** `P × Δt` за тіки 2с

Цикл оновлення: `tick()` → `aggregateStats()` → `broadcast()` через WebSocket.
