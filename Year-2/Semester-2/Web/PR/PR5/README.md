# PR5: Клієнтський веб-додаток (JavaScript + API)

**Дедлайн:** 3 травня
**Google Classroom:** [Відкрити завдання](https://classroom.google.com/u/0/c/ODE4NTM3MzA5OTg0/a/NzkyNjEyOTMwOTI1/details)

## Опис

Розробити клієнтський веб-додаток, що працює з REST API або WebSocket для завантаження та візуалізації даних енергетичного об'єкта в реальному часі.

**Вимоги:**
- Споживання REST API (fetch/axios)
- WebSocket для оновлення в реальному часі
- Візуалізація даних: графіки, таблиці, діаграми
- Бібліотеки візуалізації (Chart.js тощо)

**Оцінювання:** дизайн (20), реалізація (50), інтерфейс (15), звіт (15)

**Стек:** HTML, CSS, JavaScript, REST API, WebSocket, Chart.js
