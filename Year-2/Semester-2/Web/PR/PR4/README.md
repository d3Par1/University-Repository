# PR4: Розробка REST API для енергетичних даних

**Дедлайн:** 19 квітня
**Google Classroom:** [Відкрити завдання](https://classroom.google.com/u/0/c/ODE4NTM3MzA5OTg0/a/ODMyNTQ0NjgwMTI4/details)

## Опис

Розробити REST API для енергетичного об'єкта, що обробляє JSON-запити/відповіді для параметрів об'єкта (потужність, напруга тощо).

**Вимоги:**
- Проєктування структури API (ендпоінти)
- CRUD-операції для енергетичних даних
- Валідація запитів та обробка помилок
- Фільтри, пошук, пагінація
- Тестування та документація API

**Стек:** Node.js, Express, JSON/DB
