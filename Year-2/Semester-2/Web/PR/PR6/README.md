# PR6: Безпека веб-додатків

**Дедлайн:** 17 травня
**Google Classroom:** [Відкрити завдання](https://classroom.google.com/u/0/c/ODE4NTM3MzA5OTg0/a/ODM0NjkzODcwMzg3/details)

## Опис

Реалізувати безпеку веб-додатку: автентифікацію, управління сесіями та захист від XSS/CSRF-атак.

**Вимоги:**
- Автентифікація та авторизація (PassportJS)
- Хешування паролів (bcrypt)
- Захист API
- Валідація та санітизація даних
- Rate limiting

**Оцінювання:** функціональність (40), безпека (30), якість коду (15), документація (15)

**Стек:** Node.js, Express, PassportJS, bcrypt, JWT
