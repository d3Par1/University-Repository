# Лабораторна робота №1

## Огляд
Ця лабораторна робота присвячена основам програмування мовою C, роботі з багатофайловою компіляцією, системами збірки (Makefile, CMake), багатопоточністю та структурами даних.

---

## Компіляція та запуск

### Збірка всіх завдань
```bash
make all
```

### Очистка
```bash
make clean
```
